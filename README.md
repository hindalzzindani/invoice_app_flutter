# تطبيق الفواتير الاحترافي

تطبيق Flutter متقدم لإدارة الفواتير والمبيعات مع دعم قواعد البيانات المحلية والسحابية.

## المميزات الرئيسية

### 🧾 الشاشات الثلاث الأساسية

1. **شاشة إنشاء الفاتورة**
   - حاسبة مبيعات متقدمة
   - إضافة عدة أصناف بسهولة
   - حقول الكمية والسعر مع لوحة مفاتيح مخصصة
   - عرض فوري للإجمالي والكميات

2. **شاشة ملخص الفاتورة**
   - عرض تفصيلي لجميع الأصناف
   - حساب المصروفات المختلفة
   - حساب التسويق (40%)
   - عرض الصافي النهائي
   - حفظ الفاتورة

3. **شاشة الفواتير المحفوظة**
   - عرض جميع الفواتير
   - معلومات سريعة
   - عرض التفاصيل الكاملة
   - حذف الفواتير

## المتطلبات

- Flutter 3.5.0+
- Dart 3.5.0+
- Android SDK

## التثبيت

```bash
cd invoice_app_flutter
flutter pub get
flutter pub run build_runner build --delete-conflicting-outputs
```

## التشغيل

```bash
flutter run
```

## البناء

### بناء APK

```bash
flutter build apk --release
```

### بناء AAB

```bash
flutter build appbundle --release
```
